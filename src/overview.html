Overview of the VerCors Tool.

The central data structure of the VerCors Tool is the AST of the intermediate language COL. Currently, the following parsers are available:

  • PVL Parser, derived from the ANTLR grammar by Arend Rensink.
  • Java 7 Parser, derived from the ANTLR4 Java 7 grammar.
  • Java 8 Parser, derived from the ANTLR4 Java 8 grammar.
  • C/OpenCL/OpenMP Parser, derived from the ANTLR4 C grammar.
  • Silver Parser, as a conversion from the Viper AST.
